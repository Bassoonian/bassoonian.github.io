In some cases we find a writen <> which was presumably not pronounced. It resulted from word-final §lang_2 /u/, which never surfaces in §lang_4, unlike the originally nasal /ũ/, which did surface as /u/. For this reason, presumably unpronounced word-final <> is specifically marked using a breve in the modernised notation, i.e. <<ŭ>>.
